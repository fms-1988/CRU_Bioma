Resumo dobre os dados do (CRU)¶

O centro de pesquisa climtica (Climatic Research Unit - CRU) da universidade inglesa East Anglia cria e disponibiliza dados climticos (temperatura, precipitao, presso, etc). Esses dados podem ter frequncia mensal, escala de (0.5 x 0.5) a (10 x 10) e cobrir todo o globo.

Esse centro coleta dados de estaes climatolgicas ao redor do mundo, verifica confiabilidade dos mesmos e, para os locais sem dados, eles usam tcnicas de interpolao espacial. A metodologia est descrita no artigo:

Harris, I., Jones, P.D., Osborn, T.J. and Lister, D.H. (2014), Updated high-resolution grids of monthly climatic observations the CRU TS3.10 Dataset. Int. J. Climatol., 34: 623642. doi:10.1002/joc.3711

Depois de prontos os dados so disponibilizados no Centre for Environmental Data Analysis (CEDA), que uma organizao inglesa para criar e divulgar dados climticos de forma colaborativa. Os dados so disponibilizados em um formato que permite agrupar multiplas dimenses e variveis : no formato '.nc' --- acrnimo para: NetCDF (Network Common Data Form) ---
